## Highlights

- Rework the mobile and desktop control app into a chat-first layout that treats each session as a
  standalone conversation, with a ChatGPT-style sidebar and a full-screen transcript-first main
  view.
- Replace the previous card-heavy `首页 / 项目 / 我的` navigation flow with direct entry into the
  latest project chat, project-scoped recent conversation history, and compact secondary access to
  Git changes and file browsing.
- Compress settings into a higher-density management page focused on relay configuration, host
  status, and recent project context instead of dashboard-style summary cards.

## Included Iterations And Remediations

- Iteration v6 / mobile-first AI workspace reset, now refined into chat-first session navigation
  and compact settings management

## Operator Notes

- Relay, agent, and shared protocol behavior stay unchanged in `0.1.15`; this release is a control
  app interaction and information-architecture update.
- Existing conversations, tasks, and project discovery records remain compatible and are now
  surfaced through the new project-scoped chat sidebar instead of the prior dashboard/card layout.
