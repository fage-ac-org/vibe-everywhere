## Highlights

- Tighten the mobile chat workspace so the page stays fixed to the viewport while only the
  transcript scrolls.
- Remove repeated host, ACP, and model metadata from the primary chat header and move context into
  a compact summary strip.
- Redesign the composer into a WeChat-style single-row input with expandable execution mode and
  model controls.

## Included Changes

- Mobile control app: fixed-height chat workspace, reduced header redundancy, and a dedicated
  scrolling transcript region that behaves correctly with the bottom composer.
- Conversation turns: one-click copy support for the full assistant reply plus all stderr events
  captured for that turn.
- Repository release docs: locked release validation guidance stays aligned with the current CI and
  tag workflow expectations.
