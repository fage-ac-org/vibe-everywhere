## Highlights

- Refined the control client into a conversation-first workspace with a dedicated chat canvas and a
  persistent sidebar for relay setup, device selection, project-folder targeting, thread history,
  and file browsing.
- Split the previous oversized `Sessions` surface into smaller Vue components so future UI changes
  do not keep accumulating in one tightly coupled dashboard file.

## Included Iterations And Remediations

- Iteration v5 / Iteration 15 tranche: conversation-first threaded AI sessions follow-up UI
  restructuring
- Release packaging follow-up: repository-owned release notes for the conversation-first sidebar
  refresh

## Operator Notes

- The primary workflow now assumes operators connect once, choose a device, set a default project
  folder, and stay in long-lived conversations on the main screen.
- Device inspection, shell sessions, previews, and other secondary tools remain available, but they
  now live behind `Devices` and `Advanced` instead of occupying the main conversation surface.
- This release does not introduce new relay or agent environment variables.
